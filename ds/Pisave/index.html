Navodila:

  • 2 novi družini za fonte (font family)
  • pogruntaš kaj je njun default vrednosti fonta
  • in nastavite vsak font dvema poljubnima različnima elementoma.
  • Poleg default fontov iz družine si izberete še svojega najljubšega in ga nastavite na poljuben bločni tekstovni element
  1. Default font iz 1 družine
  2. Default font iz 1 družine ampak nek drug element
  3. najljubši font iz 1 družine
  4. Default font iz 2 družine
  5. Default font iz 2 družine ampak nek drug element
  6. najljubši font iz 2 družine
